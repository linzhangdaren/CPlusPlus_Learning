| 命令                     | 说明                                       |
|--------------------------|--------------------------------------------|
| `git add .`             | 将当前目录的所有更改添加到暂存区           |
| `git status`            | 查看当前仓库的状态，包括暂存区和工作目录   |
| `git commit -m "message"` | 提交暂存区的更改到本地仓库，并添加提交信息   |
| `git push`              | 将本地仓库的更改推送到远程仓库             |
| `git log --oneline`     | 以单行格式查看提交记录                     |
